# 🍽️ Restaurant Management System (C++ OOP)

A console-based application built to manage a restaurant's core operations. This project was developed to practically apply **Object-Oriented Programming (OOP)** concepts in C++.

## 🚀 Features (Minimum Viable Product)
* **Client Module:** Users can enter their details, delivery location, and place multiple food orders.
* **Admin Module:** Secured by a password, admins can view all incoming orders and add new restaurants to the system network.
* **Data Persistence:** New orders are logged and saved to an external text file (`OrdersLog.txt`).

## 🛠️ Technologies & Concepts Used
* **Language:** C++
* **OOP Principles:** * **Inheritance:** Base class `Clsuser` inherited by `Clsclient`, `Clsadmin`, and `Restaurant`.
    * **Polymorphism:** Utilizing virtual functions and overriding (e.g., `DisplayInfo()`).
    * **Encapsulation:** Securing data attributes using private access modifiers.
